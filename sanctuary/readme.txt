The dayz_dev directory contains the original hive.
The dayz_lingor directory contains a copy of the modified Ligor Island Sanctuary Server files for newest DayZ Lingor Server Version.
The dayz_sanctuary directory contains files to merge with new DayZ Server Files to get new "Sanctuary" to work.
The dayz_server directory contains a copy of these scripts merged with some hopefully recent version of dayz_server.pbo.
